## Description
This is a Bash shell script guessing game where the user tries to guess the number of files in the current directory.

## How to Play
1. Run the game with: `bash guessinggame.sh`
2. The program will ask you to guess how many files are in the current directory
3. Enter your guess as a number
4. The program will tell you if your guess is too high, too low, or correct
5. Keep guessing until you get the right answer!
